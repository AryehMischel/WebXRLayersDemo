DEMO: WebXR Layers with Compressed GPU Textures

A demo using the WebXR Layer API to render high quality stereo images with the help of compressed GPU textures.

This demo is mostly targeting the Meta hardware, however in the future we can easily make this demo work across devices by using a basis universal intermediate file format.
